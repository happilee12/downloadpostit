# postit
POSTit은 데스크탑에서 가장 간단하게 메모를 작성하고 관리할 수 있는 서비스입니다.
작성한 메모를 한눈에 볼 수 있고, 더이상 사용하지 않는 메모는 따로 보관할 수 있습니다. 

### 앱소개 및 다운로드
